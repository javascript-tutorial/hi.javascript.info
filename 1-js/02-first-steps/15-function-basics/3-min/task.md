# Function min(a, b)

# फंक्शन min(a, b)

Write a function `min(a,b)` which returns the least of two numbers `a` and `b`.
एक फ़ंक्शन लिखें `min(a,b)` जो कम से कम दो नंबर `a` और `b` लौटाता है।

For instance:
उदाहरण के लिए:

```js
min(2, 5) == 2
min(3, -1) == -1
min(1, 1) == 1
```
